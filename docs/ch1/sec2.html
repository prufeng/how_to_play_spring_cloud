2. 什么是Spring Boot

    Spring Boot是构建Spring应用的起点。Spring Boot的设计是为了让我们的应用能够以最少的提前配置，尽可能快地跑起来。Spring Boot坚持的一个观点是，总是构建可用于生产环境的应用程序。
    Spring Boot遵循约定优于配置的设计范式，简化了基于Spring的应用开发，大多数Spring Boot只需要很少的配置就可以跑起来。我们可以很容易地创建出可以运行的、独立的、生产级别的Sping应用。
    Spring Boot是构建Spring Cloud微服务架构的基础框架。
